HTML

Guía rápida para escribir HTML moderno y semántico: estructura, meta, texto, enlaces, listas, multimedia, tablas, formularios, accesibilidad y buenas prácticas.

Estructura básica

Plantilla mínima recomendada.


<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
</head>
</html>

DOCTYPE primero.

meta charset lo más arriba posible.

viewport para responsive.

Metaetiquetas

Básicos + SEO + redes.


<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="Descripción clara">

<meta name="robots" content="index, follow">
<link rel="canonical" href="https://ejemplo.com/pagina">

<meta property="og:title" content="Título">
<meta property="og:description" content="Descripción">
<meta property="og:image" content="https://ejemplo.com/img.jpg">

Formularios


<form action="/procesar" method="POST">

  <label for="email">Email</label>
  <input id="email" name="email" type="email" required>

  <label for="msg">Mensaje</label>
  <textarea id="msg" name="msg" rows="4"></textarea>

  <button type="submit">Enviar</button>

</form>

Siempre label + for conectado a un id.

Usa required, minlength, pattern cuando aplique.

Accesibilidad (A11y)


<a href="#main">Saltar al contenido</a>

<img src="decoracion.jpg" alt="">

  Menú
</button>

Prioriza semántica real antes que ARIA.

Mejores prácticas

  • HTML válido e indentación consistente.
  • Imágenes con alt descriptivo (o vacío si decorativa).
  • Links externos con rel="noopener noreferrer".
  • Performance: loading="lazy" en imágenes no críticas.
  • Testing: Validator, Lighthouse, WAVE / axe.
