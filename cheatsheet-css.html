CSS

Guía rápida de CSS moderno: selectores, texto, colores/fondos, box model, display/position, flexbox, grid, transiciones, animaciones y media queries.

Selectores

Base + atributo + combinadores + pseudo-clases + pseudo-elementos.


/* Básicos */
* {box-sizing: border-box; }
h1 { font-size: 2rem; }
.destacado { font-weight: 700; }
#encabezado { text-align: center; }

/* Atributo */
input[type="text"] { border: 1px solid #ccc; }
a[href*="github"] { color: #333; }
img[src^="https"] { border: 2px solid green; }
a[href$=".pdf"] { color: red; }

/* Combinadores */
article p { line-height: 1.6; } /* descendiente */
nav > ul { list-style: none; } /* hijo directo */
h2 + p { margin-top: 0; } /* adyacente */
h2 ~ p { color: #666; } /* hermanos */

/* Pseudo-clases */
a:hover { text-decoration: underline; }
input:focus { outline: 2px solid #8fd1fc; }

/* Estructurales */
li:first-child { font-weight: 700; }
tr:nth-child(even) { background: #f9fafb; }

/* Pseudo-elementos */
p::first-line { font-weight: 700; }
::selection { background: #ffda73; color: #282828; }

Regla práctica: clase para estilo, id para JS/anclas.

Prioriza selectores simples (mejor mantenibilidad).

Propiedades de texto

Fuente + formato + truncado + responsive.


.texto {
  font-family: "Lato", system-ui, sans-serif;
  font-size: 16px;
  font-weight: 400;
  line-height: 1.6;

  color: #282828;
  text-align: left;
  letter-spacing: 0.01em;
}

.titulo {
  font-size: clamp(1.6rem, 4vw, 3rem);
  font-weight: 700;
  line-height: 1.1;
}

.truncado {
  width: 220px;
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;
}

Colores y fondos

Formatos + backgrounds + gradientes + variables.


/* Formatos */
.color { color: #785082; }
.fondo { background: rgb(240, 180, 0); }
.sombra { box-shadow: 0 8px 18px rgba(0,0,0,.12); }

/* Fondo con imagen */
.hero {
  background-image: url("img.jpg");
  background-repeat: no-repeat;
  background-position: center;
  background-size: cover;
}

/* Gradiente */
.banner {
  background: linear-gradient(135deg, #785082, #6990aa);
}

/* Variables (ya las tienes en :root) */
.btn {
  color: var(--white);
  background: var(--secondary);
}

Box model

width/height + padding/margin + border/outline + shadow + overflow.


.card {
  width: min(520px, 100%);
  padding: 20px;
  margin: 20px auto;

  border: 1px solid var(--light);
  border-radius: 10px;

  box-shadow: 0 10px 24px rgba(0,0,0,.08);
  overflow: hidden;
}

/* outline (no ocupa espacio) */
.input:focus {
  outline: 2px solid var(--accent);
  outline-offset: 2px;
}

Display / Position

block/inline/inline-block + none + position + z-index.


/* display */
.oculto { display: none; }
.inline { display: inline; }
.bloque { display: block; }
.inlineblock { display: inline-block; }

/* position */
.padre { position: relative; }
.hijo {
  position: absolute;
  top: 12px;
  right: 12px;
}

.fixed-top {
  position: fixed;
  top: 0; left: 0; right: 0;
  z-index: 999;
}

.sticky {
  position: sticky;
  top: 10px;
}

Flexbox

Layout 1D, perfecto para navs, filas/columnas, gaps.


.row {
  display: flex;
  align-items: center;
  justify-content: space-between;
  gap: 16px;
}

.wrap {
  display: flex;
  flex-wrap: wrap;
  gap: 16px;
}

.item {
  flex: 1 1 240px; /* grow shrink basis */
}

.push-right {
  margin-left: auto; /* empuja a la derecha */
}

CSS Grid

Layout 2D, ideal para dashboards, galerías y cards.


.grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(260px, 1fr));
  gap: 18px;
}

.card {
  background: var(--white);
  border: 1px solid var(--light);
  border-radius: 10px;
  padding: 18px;
}

/* Areas */
.layout {
  display: grid;
  grid-template-columns: 220px 1fr;
  grid-template-areas:
    "sidebar main";
}
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }

Transformaciones y transiciones

Animar cambios sin JS (hover, focus, etc.).


.btn {
  transition: transform .15s ease, box-shadow .2s ease;
}

.btn:hover {
  transform: translateY(-2px);
  box-shadow: 0 10px 18px rgba(0,0,0,.12);
}

.card:hover {
  transform: scale(1.01);
}

Evita animar width/height/top/left (reflow). Mejor transform/opacity.

Animaciones

Keyframes + control (duración, delay, fill-mode).


@keyframes fadeUp {
  from { opacity: 0; transform: translateY(10px); }
  to   { opacity: 1; transform: translateY(0); }
}

.aparece {
  animation: fadeUp .35s ease-out both;
}

/* Accesibilidad */
@media (prefers-reduced-motion: reduce) {
  * { animation: none !important; transition: none !important; }
}

Media queries

Mobile-first + breakpoints comunes + preferencias de usuario.


/* Mobile default */
.container { padding: 16px; }

/* Tablet */
@media (min-width: 768px) {
  .container { padding: 24px; }
}

/* Desktop */
@media (min-width: 1024px) {
  .container { padding: 32px; }
}

/* Dark scheme (si quieres complementarlo a tu .dark) */
@media (prefers-color-scheme: dark) {
  :root { color-scheme: dark; }
}

Escribe primero el estilo mobile y escala hacia arriba.

Usa clamp() para tipografía responsive sin mil breakpoints.
