# 💻 Portfolio Personal – Inés Sacristán

Proyecto final del **Módulo 1 Full Stack Developer + IA** desarrollado de forma incremental a través de **6 sprints**, aplicando HTML, CSS y JavaScript modular.

## 🚀 Funcionalidades

- Diseño responsive
- Modo oscuro con persistencia (`localStorage`)
- Header y footer reutilizables
- Sección de skills dinámica
- Cheatsheets de HTML, CSS y JavaScript
- Página de recursos con sistema de tabs
- Widget de clima en tiempo real con **WeatherAPI**
- Formulario de contacto con **FormSubmit**
- Sección de proyectos con enlaces a GitHub y demo

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES Modules)
- WeatherAPI
- FormSubmit
- LocalStorage
